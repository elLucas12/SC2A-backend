# SC2A - Backend

Código do módulo _backend_ do Sistema de Controle de Assinaturas de Aplicativos, baseado na lógica de validação de aplicativos de celular.

## Instalação de Dependências

```bash
$ npm install
```

## Rodando a Aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
